# advent-of-code-2025
Solutions to puzzles on https://adventofcode.com/2025


## Getting started

```sh
go test ./day01 -v -log # To test both parts
go test ./day01 -v -run TestSolvePartOne -log # To test part 1 only
go test ./day01 -v -run TestSolvePartTwo -log # To test part 2 only
```

```sh
go run main.go --day 1 # To run both parts
go run main.go --day 1 --part 1 # To run a specific part
```
